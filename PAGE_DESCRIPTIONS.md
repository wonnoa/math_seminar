# 페이지 설명

화면에는 설명 문구를 노출하지 않고, 이 파일에서만 각 페이지의 개발 목적을 정리한다.

## 홈
- 경로: `/index.html`
- 목적: 선형대수 스터디의 메인 진입 페이지
- 핵심: 저니맵, 전체 진행도, 현재 학습 흐름

## 세션
- 경로: `/sessions.html`
- 목적: 날짜별 스터디 기록 목록
- 핵심: 세션 리스트와 개별 세션 노트 진입

## 세션 상세
- 경로:
  - `/session-2026-03-15.html`
  - `/session-2026-03-22.html`
  - `/session-2026-03-29.html`
- 목적: 개별 스터디 세션 페이지
- 핵심:
  - 지난 세션은 간단한 기록 형태로 유지
  - `2026-03-29` 페이지는 실시간 편집 노트 공간으로 사용

## 진행 상황
- 경로: `/progress.html`
- 목적: 챕터와 섹션 기준 진행 상황을 모아볼 페이지
- 핵심: 누적 진행도와 학습 깊이 신호

## 미래 지향
- 경로: `/future.html`
- 목적: 다음 세션과 앞으로의 방향을 정리할 페이지
- 핵심: 다음 주제, 열린 질문, 스터디 방향

## 멤버
- 경로: `/member.html`
- 목적: 스터디 멤버 프로필을 정리하는 페이지
- 핵심: 사진, 이름, 소개를 담은 미니 프로필 카드

## 공지사항
- 경로: `/notice.html`
- 목적: 스터디 공지 보드
- 핵심: 상단 고정 메인 공지와 날짜별 세션 공지
